# UCR-CS170-Project1_8puzzle
This is my project 1 in CS170: Introduction to AI class. This project implements three search algorithms to solve the 8-puzzle problem. The three ways are :Uniform Cost Search, A* with Misplaced Tile heuristic, and A* with Manhattan Distance heuristic.
